> 이 문서는 예전 가리사니 사이트에서 작성된 문서 입니다.
현재 상황과 맞지 않을 수 있습니다.


``` java
public static int 최대공약수(int a, int b)
{
	int tmp;
	if (a <= 0 || b <= 0) { return -1; }

	while (b != 0)
	{
		tmp = b;
		b = a % b;
		a = tmp;
	}

	return a;
}
```
``` java
public static int 최소공배수(int a, int b)
{
	if (a <= 0 || b <= 0) { return -1; }
	return a * b / 최대공약수(a, b);
}
```
