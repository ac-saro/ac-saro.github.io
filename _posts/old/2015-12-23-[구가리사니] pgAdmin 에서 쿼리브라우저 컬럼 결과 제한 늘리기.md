> 이 문서는 예전 가리사니 사이트에서 작성된 문서 입니다.
현재 상황과 맞지 않을 수 있습니다.


pgadmin result cut off 를 할 순 없지만 제한 수를 늘릴 수 는 있습니다.

1. pgAdmin 을 켭니다.
2. File -> Options... -> Query editor 에 갑니다.
3. Max. characters per column 값을 충분히 늘립니다.

이제 시원시원하게 나오는 결과를 볼 수 있습니다.
