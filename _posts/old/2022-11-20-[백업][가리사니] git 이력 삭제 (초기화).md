### 이력 삭제

1. 프로젝트 폴더로 이동
1. .git 폴더 삭제
    linux
    ```
    rm -rf .git
    ```
    windows
    ```
    rd /s /q .git
    ```
1. 초기화 후 커밋
    ```
    git init
    git add .
    git commit -m "커밋 메시지"
    ```
1. 리모트 설정
    ```
    git remote add origin <git 주소>
    ```
    예시)
    ```
    git remote add origin git@github.com:anissia-net/anissia-core.git
    ```
1. 강제 push
    ```
    git push -u --force origin master
    ```
